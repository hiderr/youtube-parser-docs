⚡ Простая установка через Docker. Работает на Windows и Mac.

Шаг 1: Установка Docker Desktop

1. Скачайте и установите Docker Desktop:

Скачать Docker Desktop

2. После установки перезагрузите компьютер

3. Запустите Docker Desktop

Шаг 2: Скачайте файл конфигурации

Скачайте файл docker-compose.yml:

Скачать docker-compose.yml

Шаг 3: Запуск приложения (Windows)

1. Нажмите Win + R, введите cmd и нажмите Enter

2. Перейдите в папку с файлом docker-compose.yml

3. Скопируйте и вставьте следующие команды:

docker-compose pull
docker-compose up -d

Шаг 3: Запуск приложения (Mac)

1. Откройте Terminal (Finder → Программы → Утилиты → Terminal)

2. Перейдите в папку с файлом docker-compose.yml

3. Скопируйте и вставьте следующие команды:

docker-compose pull
docker-compose up -d

Шаг 4: Использование

1. Откройте браузер

2. Перейдите по адресу: http://localhost:8101

Управление приложением

# Остановить приложение:
docker-compose down

# Перезапустить:
docker-compose restart

# Обновить:
docker-compose pull && docker-compose up -d
